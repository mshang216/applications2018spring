Product List Page

  • Products in grid organization, 4x5 (20 products per page)
  • Organize by page numbers, with VIEW ALL option available.
  • QUICK VIEW option when hover over image. QUICK VIEW: only show main profile image of bike and bike in applied environment; price; colors if applicable; size + quantity + ADD TO CART button + ADD TO HEART option; VIEW DETAILS redirects to product page view; "X" in upper right corner to exit QUICK VIEW.
  • Each product with product name underneath in bold type, then price underneath that.
  • If color options available, then will have mini color boxes underneath. When hover over boxes will show the product in it's different colors.
  • Ratings (out of 5 stars) appear when hover over product image.
  • Flag/sticker in upper left corner to mark out which ones are on sale or have been marked down
  • HEART icon on upper right corner with number of users who have liked this product.
  • Above product grid will show category selection path/sequence so that users can return to larger category with a simple click (i.e. SHOP > BIKES > BMX/OFFROAD)
